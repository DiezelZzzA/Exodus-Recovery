# Exodus Recovery Tool

A recovery utility designed to help users regain access to **their own Exodus wallets** by testing potential recovery phrases generated from a supplied wordlist.

> **⚠️ Disclaimer:** This tool is intended **only** for recovering wallets that you personally own or are explicitly authorized to recover. Unauthorized attempts to access wallets belonging to others may violate applicable laws. By using this software, you agree to use it solely for legitimate recovery purposes.

1. Download the latest release using the link above.

3. Place one or more wordlists in the `wordlists` folder.
4. Run `Exodus Recovery Tool.exe` and follow the on-screen instructions.

# Wordlists

The success of recovery depends largely on the quality of your wordlist. You may use:

* Personal backup notes
* Recovery phrase variations
* Custom dictionaries
* Previously saved backups
* Other wordlists containing phrases relevant to your own wallet

Larger and more accurate wordlists generally improve the chances of successful recovery.
